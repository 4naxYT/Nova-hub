## 🎯 Aimbot API

The aimbot API is a modified version of **[Exunys Aimbot API](https://github.com/Exunys/AirHub-V2)**.  
Modifications include:

- Modified drawing API for improved compatibility across executors  
- Circle formation modified  
- Optimized visibility checks  
- Whitelist/blacklist removed for debloating  

---

## ✏️ Drawing API

The drawing API is fully custom and built for reliability.  
Supported shapes include:

- Circles  
- Polygons  
- Lines  
- Squares  
- Rectangles  
- Stars  
- Triangles  
- Spirals  
- And more…
